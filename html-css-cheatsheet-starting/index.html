HTML Element & CSS Properties Cheatsheet

HTML Elements

HTML Element Name Tag Definition Self-terminating?
Embed Audio <audio> Used to embed one or more sound content sources into documents No
Bring Attention To (Bold) <b> Used to draw attention to element's contents (i.e. boldens text) No
Document Body <body> Represents content of an html document (1 used per document) No
Data Element <data> Links a given piece of content with a machine-readable translation No
Content Division <div> Generic container for flow content No

CSS Properties

CSS Property Name Property Tag Definition
Animation animation Applies an animation between styles
Border border Sets an element's border
Font font Sets all the different properties of an element's font
Margin margin Sets the margin area on all four sides of an element
Overflow overflow Sets the desired behavior when content does not fit in the parent element box in the horizontal and/or vertical direction
